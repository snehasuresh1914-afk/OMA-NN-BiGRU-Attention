# OMA-NN-BiGRU-Attention

Hybrid Deep Learning Model for Construction Cost and Schedule Prediction using OMA and Attention.

---

## Overview

This project implements a hybrid deep learning framework combining:

* Neural Network (for static features)
* BiGRU with Attention (for sequential features)
* Optical Microscope Algorithm (OMA):

  * Stage 1: Hyperparameter optimization
  * Stage 2: Output weight optimization

---

## Experiments

### Cost Prediction

Run:

```bash
python Cost_main.py
```

### Schedule Prediction

Run:
